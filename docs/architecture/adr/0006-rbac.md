# ADR-0006:RBAC 权限模型

- 状态:已接受 · 日期:2026-07-09 · 关联:REQ-701/702

## 决策
标准 RBAC:`user - role - permission`(`role_permission` 关联表)。权限校验在 **API 层用 FastAPI 依赖注入**实现;后台可视化管理角色-权限绑定,改动即时生效(权限缓存写穿/短 TTL)。

## 理由
功能权限需后台可管、可审计、可即时调整;依赖注入使校验集中、可测(behave 越权用例)。

## 后果
- 每个受保护端点声明所需 permission;无权限 → 403 + 审计日志。
- 权限判定逻辑属 domain/service,不散落在 api;越权用例是闭环①的常驻验收。

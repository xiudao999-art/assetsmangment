账号与角色

新增 / 删除账号。新账号默认普通用户;admin 是唯一管理员。

内容安全白名单

加入这些词后,即便阿里云内容安全判它违规,只要命中的词都在白名单里就放行——用来治误伤(比如引用/玩笑里的"杀人犯")。即时生效。

绝对禁词

审核第一波:物料里只要出现这里的任一词,直接拦截、不再往下判。只放非常确定不能出现的硬词(如特定政治/黄暴词);别放「去/来/广告」这类常见字,否则会误拦。即时生效。

项目管理

作品必须归属一个项目。每个项目在「审核规则」里配它自己的规则;作品要同时过标准规则 + 所属项目规则才算通过。项目下还有作品时不能删除。

功能权限

给某个用户授权功能权限,授权即时生效。点用户右侧「授权」,勾选权限后确认。

审核规则

配置违规判定:审核时把「标准 + 所属项目」规则整合成一份清单,由大模型按整篇语义判断物料是否违规(先过绝对禁词、阿里云通用拦截,最后走这里语义细判)。作用域选「标准(全局)」= 对所有物料+作品生效;选某个项目 = 只对该项目的作品额外生效。

把整篇「卡审 / 审核标准」文档粘进来,大模型按语义拆成一条条规则,落到上面选中的作用域。可先预览、删掉个别条,再确认入库。
